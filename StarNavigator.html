基本資料

探索星號是獅子級(Leo-class cruise ship)的船, 特別針對亞洲台灣人的喜好進行優化; 2019年改名.

母公司 雲頂香港(StarDream Cruises)公司, 母港在新加坡
承建商 德國 Meyer Werft
噸位 75,338 噸(星夢郵輪旗下最小噸位)
下水年代 1999年10月10日
長度 / 寬度 / 甲板層 268.6 公尺 / 32.2 公尺/ 13 層 (遊客可到 10 層樓)
房間數 928 間
載客量 1,854 (下舖)
母港 高雄, 基隆, 新加坡

2025年11月起航程:

  • 3天2夜: 高雄 -> 佬沃(Laoag) -> 高雄
  • 4天3夜: 基隆 -> 那霸 -> 石垣島 -> 基隆
  • 6天5夜: 高雄 -> 佬沃(Laoag) -> 科隆島(Coron) -> 公主港(Puerto Princesa) -> 海上巡遊 ->高雄
  • 6天5夜: 高雄 -> 佬沃(Laoag) -> 科隆島(Coron) -> 長灘島(Boracay) -> 海上巡遊 -> 高雄
